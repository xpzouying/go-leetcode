# 171. Excel表列序号


**难度：简单**


给定一个Excel表格中的列名称，返回其相应的列序号。

例如，

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
示例 1:

输入: "A"
输出: 1
示例 2:

输入: "AB"

### 思路

这个相当于一个26进制的数。

每次取最后一个字符，然后乘以一个BASE值，当前BASE为26^n。

最后一个字符C，对应的数值为：
(C - 'A') + 1
