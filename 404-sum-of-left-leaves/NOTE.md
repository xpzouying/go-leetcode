# 404. 左叶子之和 题解

## 思路

题目要求有，

1. 叶子节点

2. 左边的叶子节点


**解题想法：**

使用递归思路。

遍历到某一个节点，需要判断该节点的Left是否为叶子节点。若是为叶子节点的条件为：

```go
left := node.Left

left.Left == nil && left.Right == nil
```

如果Left为叶子，则为跳出递归条件。
